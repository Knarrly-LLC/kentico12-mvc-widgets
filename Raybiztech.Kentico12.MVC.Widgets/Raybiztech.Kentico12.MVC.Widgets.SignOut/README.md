# Sign Out Button

Displays a button that allows users to log out.

# Installation

Install the Raybiztech.Kentico12.MVC.Widgets.SignOut NuGet Package to your Kentico 12 MVC Site.

# Widget

This is a widget which allows you to add a SignoutButton to your screen with certain fields.When click on this Button User will logout form site.
